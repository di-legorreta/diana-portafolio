WORLD CLOCK APP

A lightweight, accessible world clock that displays live date and time for multiple cities with a retro pixel aesthetic.

Features
  • Four default city panels
  • Live updates every second via setInterval
  • Accurate time zones using moment-timezone
  • Responsive layout and keyboard-friendly controls

Tech
HTML5, CSS3 (Grid/Flex), JS, Moment.js and Moment Timezone
